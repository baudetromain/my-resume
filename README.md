## Development Environment

- Install [Typst](https://github.com/typst/typst)

- Install [Just](https://github.com/casey/just)

## Build Resume

```bash
just build
```

## Interactive Development Resume

```bash
just dev
```

## Containerized Build

```bash
just containerized-build
```
